about me

I am an undergraduate student in Computer Science at NMIMS, graduating in 2029.

I like figuring out how complex software actually works. Instead of just relying on pre-built libraries, I spend a lot of time writing machine learning algorithms and computer vision pipelines from scratch so I can truly understand the math behind them. When I'm not writing code, I'm usually building autonomous drones, competing in hackathons, or putting together hands-on hardware projects.

Apart from playing with computer programs, I enjoy learning about finance, Artificial Intelligence and thinking through puzzles. Outside academics, I like playing football, video games based on strategy and RPGs.

I originally hail from Kolkata, the City of Joy. Growing up there taught me patience, self-discipline, and an appreciation for consistency—values that continue to influence how I approach both work and life.

h-detection

autonomous target tracking system based on shape and contours. focuses on identifying and centering landing zones (letter h/helipads) using real-time video frames and geometric vision analysis.

stride

autonomous object detection, mapping, and obstacle avoidance system built for unmanned aerial systems (drones). integrates critical infrastructure defect inspection models.

aether

autonomous ev based project focusing on communication protocols, multi-agent coordination, and real-time obstacle reaction for electric vehicles.

project kairos

ai-powered drone surveillance for rapid crisis response. features real-time drone video streams via rtsp, intelligent detection using a yolov8s model fine-tuned on aerial datasets, automated triage classifying motionless detections as critical, and geospatial intelligence mapping camera pixel coordinates to real-world tactical maps.

resq mission control

an emergency search, rescue, and response project where significant effort was spent on fine-tuning a custom yolo detection model to track targets from drone streams rather than build standard dashboards, integrating location feeds into a terminal dispatcher interface.

titanic kaggle ensemble

ensemble prediction workflow evaluating gradient boosting, random forest, and logistic classifiers to maximize prediction performance on survival rates.

gcs interface

a custom telemetry dashboard similar to mission planner, built for the isro-iroc challenge to showcase live visual rendering of autonomous drone parameters, speeds, flight plans, and location charts.

summertracker

interactive tracker displaying productivity grids, task milestones, and visual graphs logging progress throughout the summer months.

object-detection

lightweight computer vision scripts testing pre-trained weight maps, image cropping, and threshold operations in mock video streams.
