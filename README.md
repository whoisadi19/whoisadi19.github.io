# Aditya Chowdhury - Machine Learning & Deep Learning Portfolio

This repository hosts my clean, modern, and interactive portfolio website. It effectively showcases my technical projects categorized into Serious Work, Hackathons, and Fun Experiments.

The site is designed with a premium glassmorphic theme, smooth micro-animations, and full responsiveness.

## 🚀 Features

- **Categorized Project Filter**: Switch between *All*, *Made for Serious*, *Made for Hackathon*, and *Made for Fun* dynamically.
- **Dynamic Stats & Progress Tracking**: Animated skill meter progress bars.
- **Experience Journey Timeline**: A customized connecting timeline tracking career and learning highlights.
- **Adaptive Dark / Light Themes**: Respects system preferences and stores choice locally with an elegant toggle switch.
- **Dynamic Duration Counter**: Shows active experience dynamically in years, months, and days.
- **Zero-Dependency Core**: Powered by pure semantic HTML, vanilla CSS grid/flex layouts, and lightweight native JS logic. Hosted and bundled using Vite.

## 🛠️ Technology Stack

- **Structure**: HTML5
- **Styling**: Vanilla CSS3 (Custom properties, grid systems, glassmorphism, responsive media-queries)
- **Logic**: Vanilla JavaScript ES6
- **Server/Bundler**: [Vite](https://vitejs.dev/)

## 💻 Local Setup & Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/whoisadi19/portfoliowebsite.git
   cd portfoliowebsite
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` in your web browser.

4. **Build production bundle**:
   ```bash
   npm run build
   ```
